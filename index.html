Cryptalyx

Cryptalyx is a file-sharing platform built with a Node.js backend and React frontend. Users can sign up, upload files encrypted client-side using the Web Crypto API, and share them via unique links. Files are stored on Backblaze B2, and authentication is managed through Auth0.
